35 // either the __uuidof() extension or the old-style CLSID_Foo / IID_IFoo names.
36 // If using the latter approach, the client can also choose whether to get the
37 // GUID definitions by defining the INITGUID preprocessor constant or by linking
38 // to a GUID library. This works in either C or C++.
